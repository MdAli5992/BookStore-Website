Related Objective from Python

(A) Compiled language (B) Interpreted language (C) Machine language (D) Assembly language

✅ Answer: (B) Interpreted language

Explanation: Python is an interpreted language, meaning the code is executed line by line by an interpreter at runtime. Unlike compiled languages (like C or Java), Python does not require compilation before execution, making development faster and easier.

(A) List (B) Tuple (C) Array (D) Dictionary

✅ Answer: (C) Array

Explanation: Python does not have a built-in array data type. Instead, lists are used as dynamic arrays. If a user needs an actual array-like structure, they must import the array module or use the numpy library for advanced array operations.

(A) To add elements to a list (B) To find the length of a string (C) To delete an object (D) To run a loop

✅ Answer: (B) To find the length of a string

Explanation: The len() function returns the number of elements in an object. It is commonly used with strings to find their length, but it can also be used with lists, tuples, dictionaries, and other iterable objects to count the number of elements they contain.
